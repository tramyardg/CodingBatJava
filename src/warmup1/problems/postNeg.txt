
Given 2 int values, return true if one is negative 
and one is positive. Except if the parameter 
"negative" is true, 
then 

return true only if both are negative. 


posNeg(-4, -5, true) → true	
posNeg(-4, -5, false) → false	
posNeg(-4, 5, false) → true	
posNeg(-4, 5, true) → false	
posNeg(1, 1, false) → false	
posNeg(-1, -1, false) → false
